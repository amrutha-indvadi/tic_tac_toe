# Tic-Tac-Toe Game
A simple Tic-Tac-Toe game developed in C. The game allows two players to take turns and play on a 3×3 board through the console.

# Features
- Two-player gameplay
- Console-based interface
- Winner detection
- Simple and easy-to-understand code

# Technologies Used
- C Programming Language
- Visual Studio

# How to Run
1. Open the project in Visual Studio.
2. Build and run the program.
3. Enter row and column values to make your move.
